// Core Principles — borrowed from the Ethereum Foundation mandate

CROPS is the acronym the Ethereum Foundation uses for its core mandate: Censorship Resistance, Open source and free, Privacy, and Security. crops.cash adopts the same principles as its foundation.

C
R

Censorship Resistance

No entity can block your transactions. Payments flow freely through the Curvy protocol regardless of jurisdiction or intermediary.

O

Open Source

Every line of code is public. Verify the protocol yourself. Trust is earned through transparency, not promises.

P

Privacy

Stealth addresses and sparse Merkle trees break the link between your identity and your payments. Even when unshielding, the source is untraceable.

S

Security

Wallet keys stay local. Zero-knowledge proofs validate without revealing. The protocol is designed so that trust is never required.

// Privacy Model

Shielded Balances

Funds inside Curvy are encrypted Notes. Only you can decrypt and see your balance. No observer can determine how much you hold or in what currency.

Private Transfers

Sending between Curvy users is fully private. Sender, recipient, amount, and currency are all hidden behind zero-knowledge proofs.

Untraceable Unshielding

When funds leave Curvy to a public address, observers see the destination but cannot trace where the funds came from. The link to you is broken.

// How it works with AgentCash

AgentCash makes x402 micropayments to premium APIs. Those payments are transparent. crops.cash makes them private.

001
Fund your Curvy wallet — deposits are automatically shielded on receipt. Your balance becomes invisible.
|
002
Agent needs to call a paid API — crops.cash unshields the required USDC to the x402 payment address.
|
003
API provider receives payment — but cannot link it back to your identity. The source of funds is untraceable.
// Commands
onboard

Create your shielded wallet. Opens browser for Curvy auth, saves wallet locally at ~/.crops.cash/wallet.json

balance

View your shielded balances across all networks. Only you can see these — they're encrypted on-chain.

send

Send funds privately. To a .curvy.name for full privacy, or to a hex address for untraceable unshielding.

info

List all supported networks and currencies. Check compatibility before initiating a transfer.
